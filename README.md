# PySchwarzschild-Geodesics
Simulador de geodésicas em buracos negros de Schwarzschild. Implementação computacional de Relatividade Geral para análise de órbitas e lentes gravitacionais.

Este projeto implementa numericamente geodésicas temporais e nulas na métrica de Schwarzschild, com foco em órbitas relativísticas, estabilidade e fenômenos de lente gravitacional.
